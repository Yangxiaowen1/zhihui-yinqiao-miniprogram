# 智汇银桥 · 微信小程序

根据项目书与设计图实现的银龄服务小程序，包含三大板块与首页、聚合、信用、我的等模块。

## 功能结构

### 1️⃣ 智享学堂（老年人给小孩上课）
- **选课程**：课程列表、分类、课程详情  
- **预约**：填写姓名、电话、备注提交预约  
- **扫码**：现场扫码签到  
- **评价**：课程结束后星级+文字评价  

### 2️⃣ 银龄智库（老年人咨询问题）
- **展示专家及预约时间**：专家列表、详情、可预约时段选择  
- **预约并提交问题**：选择时段后填写咨询问题并提交  
- **记录结果**：咨询记录页展示历史咨询与结果  

### 3️⃣ 青银共创（老年人与青年人组队做活动）
- **活动发布**：活动列表与详情展示  
- **报名**：填写姓名、电话、身份（银龄/青年）报名  
- **组队**：一键匹配队友或分享邀请好友组队  

### 其他
- **首页**：搜索栏、Banner（银龄智库专家巡讲）、三大入口、我的报名 / 银龄信用档案 / 消息通知  
- **聚合**：课程、专家、活动快捷入口  
- **信用**：信用分与银龄信用档案入口  
- **我的**：我的报名、信用档案、消息通知  
- **搜索**：按课程 / 专家 / 活动搜索  

## 如何运行

1. 安装并打开 **微信开发者工具**  
2. 选择「小程序」→「导入项目」  
3. 目录选择本项目的 **miniprogram 文件夹**（`c:\Users\zhouj\Desktop\aaa\miniprogram`，即包含 `app.json` 的目录）。  
4. AppID 可先选「测试号」或「游客模式」进行预览。  
5. 编译后即可在模拟器或真机预览。

## 目录说明

- `app.js` / `app.json` / `app.wxss`：小程序入口与全局配置  
- `custom-tab-bar`：自定义底部导航（首页 / 聚合 / 信用 / 我的）  
- `pages/`：各页面  
  - `index`：首页  
  - `course/*`：智享学堂（list / detail / book / scan / evaluate）  
  - `expert/*`：银龄智库（list / detail / book / record）  
  - `activity/*`：青银共创（list / detail / register / team）  
  - `aggregate` / `credit` / `mine`：底部 Tab 页  
  - `my-registrations` / `credit-archive` / `notifications` / `search`：通用功能页  

当前为前端演示版本，列表与详情数据为本地模拟；接入真实后端时，在对应页面的 `onLoad` / 列表方法中替换为 `wx.request` 等接口调用即可。
